Getting Started with Nox PHP

Nox aims to introduce no new syntax other than a very basic markup for view files. Nox is a PHP-first-and-only web framework that aims to have a small footprint on your projects while introducing an easy-to-setup MVC architecture.

Nox makes no attempts to be a full-service web framework, but instead serves to assist in creating your own apps and projects by being a skeleton to canvas your custom flair onto. Nox provides standard MVC attribute routing, a view file engine, static file serving, a lightweight MySQL ORM, and configurable project JSON files.

This framework is not a web server and is currently only compatible with Apache web servers.
